1. Funci칩n de Supervivencia \(S(t)\)

  • Define la probabilidad de sobrevivir m치s all치 del tiempo \(t\):
    \[ S(t) = P(T > t) \]
  • Es mon칩tona decreciente
  • Se relaciona con la funci칩n de distribuci칩n:

2. Funci칩n de Riesgo \(h(t)\)

  • Tasa instant치nea de falla en \(t\) dado que se ha sobrevivido hasta \(t\):
    \[ h(t) = \lim_{\delta \to 0} \frac{P(t < T < t+\delta \mid T > t)}{\delta} \]
  • Tambi칠n llamada tasa de falla o funci칩n de intensidad

3. Relaci칩n entre funciones

  • \[ f(t) = -\frac{d}{dt}S(t) \]
  • \[ h(t) = \frac{f(t)}{S(t)} \]
  • \[ S(t) = \exp\left(-\int_0^t h(u) du \right) \]

4. Distribuciones Param칠tricas

  • Exponencial:
    • \(h(t) = \lambda\) constante
    • \(S(t) = e^{-\lambda t}\)
  • Weibull:
    • \(h(t) = \alpha \lambda^\alpha t^{\alpha - 1}\)

  • Gamma y otras

Media y mediana en modelos param칠tricos

Modelo Exponencial

  • Si \(T \sim \text{Exp}(\lambda)\):

    • Media:
      \[ E(T) = \frac{1}{\lambda} \]

    • Mediana:
      \[ \text{Mediana} = \frac{\ln(2)}{\lambda} \]

lambda <- 0.1
media_exp <- 1 / lambda
mediana_exp <- log(2) / lambda
media_exp; mediana_exp
[1] 10
[1] 6.931472

Modelo Weibull

  • Si \(T \sim \text{Weibull}(\alpha, \lambda)\):

    • Media:
      \[ E(T) = \lambda^{-1} \cdot \Gamma\left(1 + \frac{1}{\alpha} \right) \]

    • Mediana:
      \[\text{Mediana} = \left( \frac{\ln(2)^{1/\alpha}}{\lambda} \right)\]

alpha <- 1.5; lambda <- 0.03
media_weibull <- (1/lambda) * gamma(1 + 1/alpha)
mediana_weibull <- (log(2)^(1/alpha))/lambda
media_weibull; mediana_weibull
[1] 30.09151
[1] 26.10733
